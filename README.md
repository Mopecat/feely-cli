# feely-cli

简单的前端构建工具

## Install

```
  npm install feely-cli -g
```

## Explain

安装后执行

```
feely create
```

后面跟着提示走就可以了~~
目前只有 ts 模板和 vue 模板，后续还会继续增加，以及丰富 `feely-cli` 的功能，有 `idea` 的同学可以 `issues` 里提一下~
模板放在 https://github.com/Mopecat/template 这个仓库里.后续将陆续添加 `react` 的项目模板
